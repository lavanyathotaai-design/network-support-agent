# AI Network Support Agent

An AI-powered network troubleshooting assistant built with Python, Streamlit, LangChain, Ollama, and ChromaDB.

## Installation

```bash
pip install -r requirements.txt
ollama pull llama3
ollama pull nomic-embed-text
python ingest.py
streamlit run app.py
```

## Sample Questions

- What does HTTP 503 mean?
- How do I troubleshoot DNS issues?
- Why am I getting SSL certificate errors?
